# Project Overview

This project demonstrates the deployment of a **full stack web application** using modern DevOps practices.

The application is deployed inside a **Virtual Machine** using **Docker containers**, and is exposed to the internet using **Cloudflare Tunnel** without requiring a public IP address.

The project combines several important DevOps concepts including **virtualization, containerization, reverse proxy configuration, and cloud service integration**.

---

# Key Features

- Full stack web application deployment
- Docker containerization for frontend and backend services
- Virtual machine infrastructure using VirtualBox
- Reverse proxy configuration using Nginx
- Secure public access without requiring a public IP address
- Integration with multiple cloud services and APIs

---

# Technologies Used

## Frontend

- React
- Vite
- Nginx

The frontend provides the user interface and communicates with the backend through API requests.

---

## Backend

- Node.js
- Express.js

The backend handles application logic, API endpoints, and integration with external services.

---

## Infrastructure

- Docker
- Docker Compose
- Ubuntu Server
- VirtualBox

Docker containers are used to run the application services inside an Ubuntu virtual machine.

---

## Cloud Services

- MongoDB Atlas (Database)
- Cloudinary (Media storage)
- HuggingFace API (AI model access)
- OpenAI API (AI features)
- Cloudflare Tunnel (Public internet access)

These services extend the capabilities of the application and enable advanced functionality.

---

# Project Goals

The main objectives of this project are:

- Demonstrate containerized application deployment
- Implement virtualization-based infrastructure
- Enable secure public access without a public IP
- Integrate external cloud services and APIs
- Showcase practical DevOps deployment techniques

---

# Summary

This project demonstrates how a modern full stack application can be deployed using **virtual machines, Docker containers, and cloud services**, while securely exposing the application to the internet using **Cloudflare Tunnel**.
